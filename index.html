Propuesta de Proyecto

a) Empresa seleccionada

TecnoSoluciones S.A.

Empresa de tecnología con más de 200 empleados que actualmente gestiona sus recursos humanos con sistemas desconectados y procesos manuales.

b) Nombre del proyecto

HRM System Pro

Sistema integrado de gestión de recursos humanos para medianas empresas.

c) Descripción del proyecto

HRM System Pro es una solución de software integral diseñada para automatizar y optimizar todos los procesos de gestión de recursos humanos en TecnoSoluciones S.A.

El sistema centralizará la información de empleados, gestionará nóminas, controlará asistencia, administrará beneficios, facilitará la evaluación del desempeño y mejorará la comunicación interna.

Gestión centralizada

d) Objetivos del proyecto

Eficiencia operativa

Reducir en un 40% el tiempo dedicado a procesos administrativos de RRHH.

Centralización de datos

Unificar toda la información de empleados en una sola plataforma.

Cumplimiento legal

Garantizar el cumplimiento de normativas laborales y de protección de datos.

Experiencia del empleado

Mejorar la satisfacción de los empleados con herramientas de autogestión.

Análisis: Principios de Diseño y Módulos HRM

Principios fundamentales de diseño en ingeniería de software

Principio de abstracción

Simplificar sistemas complejos ocultando detalles irrelevantes y mostrando solo características esenciales.

Principio de modularidad

Dividir el sistema en componentes independientes con interfaces bien definidas para facilitar mantenimiento y escalabilidad.

Principio de encapsulamiento

Ocultar los detalles de implementación y exponer solo interfaces necesarias para interactuar con el componente.

Principio de cohesión

Cada módulo debe tener una única responsabilidad bien definida y relacionada.

Principio de bajo acoplamiento

Minimizar las dependencias entre módulos para facilitar cambios independientes.

Descripción detallada de módulos:

  • Gestión de empleados: Base de datos centralizada con información personal, profesional y contractual.
  • Nómina: Automatización de cálculos salariales, deducciones, impuestos y generación de recibos.
  • Asistencia: Control de horarios, vacaciones, permisos y ausencias con integración a dispositivos biométricos.
  • Reclutamiento: Gestión de vacantes, candidatos, entrevistas y procesos de selección.
